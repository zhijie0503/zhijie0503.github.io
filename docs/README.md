## 游戏概述

方格消除游戏是一个基于React和TypeScript的网页小游戏，玩家通过拖动鼠标选择相邻的方格，当选中方格的数字总和等于10时，这些方格会被消除并重新生成，玩家获得相应的分数。

游戏特点：
- 简单直观的游戏规则
- 流畅的拖动选择操作
- 精美的视觉效果和动画
- 实时的游戏状态反馈
- 清晰的得分显示

## 快速指南

要开始开发或玩这个游戏：

```bash
# 克隆仓库
git clone [仓库地址]

# 进入项目目录
cd games-demo

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

默认情况下，游戏将在 http://localhost:5173/ 运行。
