# 方格消除游戏 - 技术栈

## 核心技术

本项目使用以下技术栈进行开发：

1. **React 18** - 用于构建用户界面的JavaScript库
2. **TypeScript** - 为JavaScript添加静态类型定义
3. **Vite** - 现代前端构建工具，提供快速的开发环境
4. **Styled Components** - CSS-in-JS解决方案，用于组件样式定义
